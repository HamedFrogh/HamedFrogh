### 🔬 What I'm working on

- **TrainDetection** — Lightweight ML on STM32F103C8T6: sliding-window feature engineering, int8-quantized TTA regression, full pipeline from raw sensor data to deployment
- **SQIsign on Pico 2** — Porting post-quantum cryptography (SQIsign) to Raspberry Pi Pico 2 (Cortex-M33): bare-metal HAL, custom linker scripts, Arduino IDE with arduino-pico BSP
- **FL-IDS** — Federated learning intrusion detection for edge IoT: Byzantine poisoning defense (AutoGM, ZBDU), non-IID heterogeneity, adaptive client selection
